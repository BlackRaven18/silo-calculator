# 🌾 Silo Capacity Calculator

A premium, professional Flutter application designed for farmers, agricultural managers, and agribusinesses. This tool provides accurate calculations for silo capacity and grain tonnage, helping users optimize their storage management with ease.

## ✨ Key Features

- **Precision Calculations**: Calculate total volume and grain mass for cylindrical silos with cone hoppers.
- **Dynamic Crop Database**: Includes a pre-configured dictionary for popular crops (Wheat, Maize, Canola, Pellets, etc.) with the ability to set custom densities.
- **Global Localization**: Fully localized in **6 languages**:
  - 🇬🇧 English
  - 🇵🇱 Polish
  - 🇪🇸 Spanish
  - 🇩🇪 German
  - 🇺🇦 Ukrainian
  - 🇵🇹 Portuguese
- **Profile Management**: Save, rename, and manage multiple silo configurations.
- **Visual Feedback**: Real-time simulation of silo fill levels with interactive controls.
- **Premium UI/UX**: Stunning modern design with support for both **Dark** and **Light** modes.
- **Cross-Platform**: Optimized for Web.

## 📦 Getting Started

### Prerequisites

- Flutter SDK (version 3.7.0 or higher)
- Android Studio / VS Code with Flutter extensions

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/BlackRaven18/silo-calculator.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd silo-calculator
   ```

3. **Install dependencies:**
   ```bash
   flutter pub get
   ```

4. **Generate localization files:**
   ```bash
   flutter gen-l10n
   ```

5. **Run the application:**
   ```bash
   flutter run
   ```
